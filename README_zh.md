# 简易时钟

[English](README.md) | 简体中文
> 逝者如斯夫，不舍昼夜。

### 项目概览

一个使用 C++ 开发的简易时钟，开发环境为 Visual Studio 2022  
<strong>外部库</strong>：[Raylib](https://www.raylib.com/)

### 特性

- <strong>利用系统时间进行实时时钟同步</strong>
- <strong>深色背景搭配白色钟面，提供强烈对比度与最大可读性</strong>
- <strong>基于面向对象原则构建，代码结构清晰，方便调整参数</strong>
