# Простые часы

[English](README.md) | [简体中文](README_zh.md) | Русский
> 逝者如斯夫，不舍昼夜。

### Обзор

Простые часы, разработанные на C++ с использованием Visual Studio 2022  
<strong>Внешняя библиотека</strong>：[Raylib](https://www.raylib.com/)

### Особенности

- <strong>Синхронизация часов в реальном времени с использованием системного времени</strong>
- <strong>Черный фон и белый циферблат обеспечивают высокую контрастность и максимальную читаемость</strong>
- <strong>Создан с использованием принципов OOP, что обеспечивает чистую и модульную структуру для настройки параметров</strong>
